Win12 Copilot（下称"本服务"）由 Win12-Online 团队（下称"我们"）所有和负责运营。我们深知个人信息对您的重要性，并始终致力于保护您的个人信息安全。本隐私政策（下称"本政策"）旨在向您说明我们如何收集、使用、存储、传输和保护您的个人信息，以及您享有的相关权利。为引起您的注意，本政策的重点部分已使用下划线特别标注。

若您是未满 14 周岁的未成年人，您不得使用本服务；若您正在使用，请您立即停止使用，并请您的法定监护人联系我们，以便我们及时处理。若您是已满 14 周岁但未满 18 周岁的未成年人，请与您的法定监护人共同阅读本政策，并在征得法定监护人同意后方可使用本服务。

本政策是《Win12 Copilot 用户协议》不可分割的组成部分。 在使用本服务前，请您仔细阅读并充分理解本政策的全部内容。如您不同意本政策的任何条款，请您立即停止使用本服务。

一、我们如何收集、使用您的个人信息

1.1 我们不主动收集您的个人信息

本服务为匿名可用，您无需注册账号、提供邮箱或任何身份信息即可使用。我们不会要求您提供姓名、手机号码、电子邮箱地址或其他可识别您个人身份的信息。

本服务是一款开源前端应用，AI 对话功能通过用户浏览器直接向第三方服务商发起请求。我们自身不运营后端服务器，不存储您的对话记录、输入内容或 AI 生成输出，亦不会将您的任何数据用于模型训练或其他分析用途。

1.2 本服务托管平台可能自动收集的信息

本服务的静态文件（HTML、JavaScript、CSS 等）托管于以下第三方平台。当您访问本服务时，这些平台可能自动收集下列技术信息：

托管平台可能收集的信息
Cloudflare Pages访问者 IP 地址、User-Agent（浏览器类型、操作系统）、请求时间、访问的页面 URL 等 HTTP 请求日志
GitHub Pages访问者 IP 地址（用于安全目的）、浏览器类型等日志信息

上述信息的收集、处理和存储由 Cloudflare, Inc. 和 GitHub, Inc. 分别依据其各自的隐私政策独立进行，我们无法访问、控制或删除这些日志数据。详情请参阅：

  • Cloudflare 隐私政策：https://www.cloudflare.com/zh-cn/privacypolicy/
  • GitHub 隐私声明：https://docs.github.com/zh/site-policy/privacy-policies/github-general-privacy-statement

1.3 我们不使用分析或跟踪工具

本服务未集成任何第三方统计代码（包括但不限于 Google Analytics、百度统计、Cloudflare Analytics），亦不使用 Cookie 或类似跟踪技术来识别或跟踪您的行为。

二、AI 对话功能的数据处理

2.1 您的输入将被传输至第三方服务商

本服务的 AI 对话功能通过第三方 API 服务商「云智 API」（yunzhiapi.cn，运营方：云智计算）调用底层大语言模型实现。您在对话界面中输入的文字内容（prompt）将在您的浏览器中直接发送至云智 API 的服务器进行处理，以生成 AI 回复。

2.2 第三方服务商的数据处理

云智 API 作为数据处理方，其用户协议（https://yunzhiapi.cn/?page=terms）赋予其收集用户的调用数据、请求内容及生成结果的权利，并可能将该等数据用于模型训练和改进。您在使用 AI 对话功能前，应阅读并了解云智 API 的用户协议与隐私政策。

2.3 重要警示：请勿提交敏感信息

鉴于上述第三方数据处理机制，我们郑重提醒您：

  • 请勿通过本服务提交任何您不希望被第三方服务商服务器处理或留存的任何信息，包括但不限于：国家秘密、商业秘密、未公开的技术方案、个人隐私数据、敏感个人信息（如身份证号码、银行账户信息、医疗健康信息、行踪轨迹等）。
  • 我们无法控制第三方服务商对其接收的数据的处理行为，在适用法律允许的最大范围内，我们对此类第三方的数据收集、使用、存储行为不承担责任。

三、信息的存储与安全

3.1 我们自身的数据存储

我们不为本服务运营后端数据库。您的对话输入和 AI 生成输出仅存在于您当前浏览器的会话中，关闭页面或会话结束后数据即从浏览器内存中清除。我们不保留您的对话记录的任何副本。

3.2 第三方存储

  • 托管平台日志：Cloudflare 和 GitHub 的日志保存期限分别依据其各自的隐私政策。据公开信息，GitHub 保存访问者 IP 日志以维护服务安全和完整性；Cloudflare 日志的典型保存期限因产品和服务而异。
  • 云智 API：您的 prompt 数据由云智 API 根据其用户协议及隐私政策进行处理和留存，我们无法获悉或控制其具体存储期限。

3.3 安全措施

尽管我们自身不存储您的个人信息，我们仍重视本服务的前端代码安全，并通过托管平台提供的 HTTPS 加密传输保障您与服务器之间的通信安全。但我们无法保证第三方服务商（托管平台、API 服务商）的安全措施，也不对此承担责任。

四、您的权利

4.1 查询、更正与删除

由于本服务不要求注册且我们不收集、不存储您的个人信息，我们通常无法识别您的个人身份，亦无法提供基于账号的查询、更正或删除服务。

4.2 关于第三方收集的信息

对于托管平台（Cloudflare、GitHub）和 API 服务商（云智 API）所持有的与您相关的信息，如需行使访问、更正、删除等权利，请您直接依据其各自的隐私政策向其提出请求。

五、未成年人保护

5.1 本服务不以未成年人为目标人群。若您是未成年人，请不要自行向我们提供您的任何个人信息或数据。

5.2 若您作为未满 14 周岁的未成年人，请勿使用本服务；若您正在使用，请立即停止使用。

5.3 若您作为已满 14 周岁但未满 18 周岁的未成年人，请与您的法定监护人共同阅读本政策，并在征得法定监护人同意后方可使用本服务；否则，请勿使用本服务。

5.4 未成年人父母或其他法定监护人如对本服务涉及未成年人个人信息的事项有任何疑问，可通过本政策第六章所列的联系方式与我们联系。
